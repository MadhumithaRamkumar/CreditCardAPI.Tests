This Project contains the unit tests for Credit Card API Project. It validates the API endpoint, service logic using xUnit.

**FrameWorks**

XUnit - Unit testing framework
.Net 9 Framework - C# .NET Framework

**SETUP INSTRUCTION**

1.  Open Visual Studio.
2.  Click on "Clone on repository".
3.  Paste the Git repo URL in the Repository location field.
4.  Choose a local path where the repo will be saved.
5.  Select Clone
